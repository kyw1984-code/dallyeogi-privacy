🔒 달력이는 모든 건강 데이터를 사용자의 기기에만 저장합니다. 서버로 전송되지 않으며, 회원가입이 필요 없습니다.

1. 수집하는 정보

달력이는 앱 사용을 위해 아래 정보를 기기 내에서만 처리합니다.

  • 생리 시작일 및 종료일
  • 증상 기록 (생리량, 통증, 신체 증상, 기분)
  • 날짜별 메모
  • 알림 설정 (D-3, D-1 알림 여부)
  • 앱 환경설정 (다크모드, 주기 길이 등)

위 데이터는 모두 사용자 기기의 로컬 데이터베이스(SQLite)에 저장되며, 외부 서버나 클라우드에 전송되지 않습니다.

2. 수집하지 않는 정보

  • 이름, 이메일, 전화번호 등 개인 식별 정보
  • 위치 정보
  • 연락처, 사진, 카메라 접근
  • 광고 식별자 (IDFA) — 맞춤 광고 미사용
  • 기기 고유 식별자

3. 정보 이용 목적

수집된 데이터는 다음 목적으로만 사용됩니다.

  • 생리 주기 예측 및 가임기 계산
  • 생리 예정일 알림 발송
  • 통계 화면 표시
  • 사용자가 직접 요청한 데이터 백업/복원

4. 제3자 제공

달력이는 사용자의 건강 데이터를 어떠한 제3자에게도 판매하거나 제공하지 않습니다.

단, 앱 내 광고 표시를 위해 Google AdMob SDK를 사용합니다. AdMob은 광고 식별자를 사용할 수 있으며, 이는 Google의 개인정보처리방침을 따릅니다.

  • Google 개인정보처리방침 →

5. 광고 (Google AdMob)

달력이는 무료 서비스 유지를 위해 배너 광고를 표시합니다.

  • 표시 광고: 배너 광고 (홈, 통계 화면 하단)
  • 맞춤 광고: 미사용 (비개인화 광고만 표시)
  • 광고 데이터는 Google이 자체 정책에 따라 처리

6. 데이터 보관 및 삭제

모든 데이터는 사용자 기기에만 저장됩니다.

  • 삭제 방법: 앱 삭제 시 모든 데이터가 즉시 삭제됩니다
  • 백업: 사용자가 직접 JSON 파일로 내보내거나 가져올 수 있습니다
  • 서버 데이터: 존재하지 않으므로 삭제 요청이 불필요합니다

7. 알림 권한

생리 예정일 알림을 위해 알림 권한을 요청합니다. 알림 권한은 기기의 설정 앱에서 언제든지 철회할 수 있습니다. 알림을 거부해도 앱의 모든 핵심 기능을 사용할 수 있습니다.

8. 아동 보호

달력이는 13세 미만 아동을 대상으로 하지 않습니다. 13세 미만 아동의 개인정보를 의도적으로 수집하지 않습니다.

9. 방침 변경

개인정보처리방침이 변경될 경우, 이 페이지를 통해 공지합니다. 중요한 변경사항은 앱 업데이트 노트에도 안내됩니다.
